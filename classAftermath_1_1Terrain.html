Detailed Description

A Terrain object represents a particular class of Tile, e.g.

Mountains, Hills, or Ocean. Terrain objects are immutable and reused for different tiles.

Parameters:

isLand- Is this terrain a land type.
isSea- Is this terrain a sea type.
image- The image of this terrain type, to be used by all tiles of the same type.
probabilities- A map between resource types and their probabilities for this tile. This is used when randomly generating resources for tiles.
